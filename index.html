TLS MITM Detector tool v.0001

About the tool

This tool scans the files within lookup.txt and compares the fingerprint of the certificate you receive locally to the fingerprint that is obtained remotely. If there are any differences it could be a result of a MITM attack where your traffic is being actively decrypted. This tool is super de-duper alpha and google.com is sometimes falsely flagged. There are no false negatives but false positives are possible.

How to use

Currently the only way to update the sites to get a fingerprint for is by editing the "lookup.txt" file. Add/remove entries from there and then restart the app.
